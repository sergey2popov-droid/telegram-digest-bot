# roadmap-step

Use this skill whenever the user asks to implement a roadmap phase.

Rules:
- Work on exactly one roadmap step at a time.
- Do not implement future steps.
- Before coding, restate the current step goal.
- After coding:
  - update ROADMAP.md
  - summarize changed files
  - list what remains stubbed
  - provide verification commands
- If the request spans multiple steps, ask to split it.
